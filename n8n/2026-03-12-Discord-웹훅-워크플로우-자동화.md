### Context
Discord 트리거를 이용하여 AI 에이전트와 통신하는 자동화를 구축하려 했으나, 윈도우 환경에서 path 이슈로 인해 Discord 트리거 설치가 불가능했습니다. 이를 해결하기 위해 Discord 봇과 웹훅을 연결하여 메시지를 수신하면 n8n 웹훅을 호출해 AI 에이전트를 실행하는 워크플로우를 설계했습니다.

### Core
1. Discord에서 메시지를 수신하면 봇이 n8n의 웹훅을 호출합니다.
2. n8n에서 AI 에이전트가 실행됩니다.
3. 에이전트는 TIL 형식으로 변환된 내용을 생성하고 이를 GitHub에 push합니다.
4. GitHub에 push가 완료되면, README.md 파일에 인덱스가 추가됩니다.
5. 완료 메시지를 발신하여 프로세스를 마칩니다.

```javascript
// 예시 코드
// Discord Webhook to n8n
"webhook":"https://n8n.example/webhook",

// n8n Execution
{
   "trigger": "Webhook Call",
   "action": "Execute AI Agent"
}

// GitHub Push
{
   "action": "Push to GitHub",
   "file": "til.md"
}
```

### Insight
몇 달 전에 사용했던 make 자동화보다 n8n이 더 전문적인 업무에 적합하다는 것을 느꼈습니다. n8n은 직관적인 인터페이스로 다양한 영역에 쉽게 적용할 수 있습니다. 각 노드의 입력과 출력을 정확히 이해하면 워크플로우 제작이 더욱 수월해집니다.
