### Context
n8n 플랫폼을 활용하여 기존의 GPT API 대신 제미나이 API로 전환함으로써 비용을 줄이고자 했으나, 무료 사용 한계로 인해 제한이 자주 발생하는 문제가 있었습니다. 이에 따라 groq라는 새로운 AI API를 활용하여 자동화를 시도했습니다.

### Core
```javascript
const triggerNode = {
  trigger: { type: 'webhook' },
};

$workflows = [
  {
    nodes: [
      {...triggerNode},
      {
        name: 'groqAPI',
        type: 'httpRequest',
        params: {
          method: 'POST',
          url: 'https://api.groq.io/v1/execute',
          body: {
            query: 'YOUR_QUERY_HERE',
          },
          headers: {
            'Content-Type': 'application/json',
          },
        },
      },
    ],
  },
];
```

### Insight
제미나이 API는 사용 빈도에 따라 제한이 있는 반면, groq API는 좀 더 유연하게 요청을 처리할 수 있게 설계되었습니다. 이러한 전환으로 인해 자동화 에이전트를 구축하는 데 있어 더 높은 효율성을 기대할 수 있습니다. 실제 테스트 결과, 요청 처리 시간이 개선되었고, 안정적인 성능을 확인할 수 있었습니다.
